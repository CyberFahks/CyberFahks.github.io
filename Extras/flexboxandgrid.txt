Flexbox Classes:
From Bootstrap:

    .d-flex: Applies display: flex to an element.
    .flex-row: Aligns flex items in a row (default).
    .flex-column: Aligns flex items in a column.
    .justify-content-start: Aligns items to the start of the container.
    .justify-content-center: Centers items horizontally.
    .justify-content-end: Aligns items to the end of the container.
    .justify-content-between: Distributes items with space between them.
    .justify-content-around: Distributes items with space around them.
    .align-items-start: Aligns items to the start of the container (vertically).
    .align-items-center: Centers items vertically.
    .align-items-end: Aligns items to the end of the container (vertically).
    .align-items-baseline: Aligns items along the baseline of the container.
    .align-self-start: Aligns an individual item to the start of the container.
    .align-self-center: Centers an individual item vertically.
    .align-self-end: Aligns an individual item to the end of the container.
    .flex-wrap: Allows flex items to wrap onto multiple lines.
    .flex-nowrap: Prevents flex items from wrapping.

Custom Flexbox Classes:

    .flex: Sets display: flex on an element.
    .flex1: Sets flex: 1 to allow an element to grow and shrink as needed.
    .flex-grow-1: Sets flex-grow: 1 to allow an element to grow.
    .flex-shrink-0: Sets flex-shrink: 0 to prevent an element from shrinking.
    .align-items-baseline: Aligns items to the baseline of the container.

Grid Classes:
From Bootstrap:

    .container: Creates a responsive fixed-width container.
    .container-fluid: Creates a full-width container.
    .row: Defines a row within the grid.
    .col: Defines a column that spans the full width of the row.
    .col-{breakpoint}-{size}: Defines a column with a specific width (e.g., .col-md-6).
    .col-auto: Allows the column to size itself based on its content.
    .col-{breakpoint}-{size}-offset: Adds an offset to a column (e.g., .col-md-6-offset-3).
    .g-{size}: Adds gap between grid items (e.g., .g-3).
    .align-items-start: Aligns items to the start of the container.
    .align-items-center: Centers items vertically.
    .align-items-end: Aligns items to the end of the container.
    .justify-content-start: Aligns items to the start of the row.
    .justify-content-center: Centers items horizontally.
    .justify-content-end: Aligns items to the end of the row.
    .justify-content-between: Distributes items with space between them.
    .justify-content-around: Distributes items with space around them.

Custom Grid Classes:

    .grid-container: Applies display: grid to an element.
    .grid-item: Applies styling to individual grid items.
    .grid-template-columns: Defines column sizes (e.g., grid-template-columns: 1fr 2fr;).
    .grid-template-rows: Defines row sizes (e.g., grid-template-rows: 100px auto;).
    .grid-gap: Defines the gap between grid items (e.g., grid-gap: 10px;).
    .grid-area: Defines named areas for grid items.
